1.1.2. Using Cons

cons - basically a node(you can put two things in it)

EX: (cons 3 ’())

note - “’()” refers to empty or null

now from before…

(cons “apple” (cons “cow” (cons “cabbage” ’())))

1.1.3. Getting elements out of Cons

  • Using first
    • (first (cons 3 ’())) -> 3
  • Using rest
    • (rest (cons 3’()) -> ’()
